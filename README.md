> 集成了 select, calendar, editor, upload, validator 等通用表单插件的 [nd-form](https://github.com/ndfront/nd-form)

## 安装

```bash
$ npm install nd-form-extra --save
```

## 使用

```js
var FormExtra = require('nd-form-extra');
// use FormExtra
```
